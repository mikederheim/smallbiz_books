SmallBiz Books update: login-required user creation

Changed files:
- app/Core/Auth.php
- app/Controllers/AuthController.php
- app/Views/layouts/register.php
- app/Views/layouts/main.php
- public/index.php

Install:
1. Back up your files and database.
2. Copy the changed files into your existing smallbiz_books install, preserving paths.
3. No MySQL changes are required.

Behavior change:
- /public/index.php?r=register is public only when there are zero users in the database.
- After the first user exists, only logged-in users can access the create-user screen.
- Logged-in users can create additional users from the new Add User link in the sidebar.
- Creating an additional user does not log you out or switch you into that new user.
