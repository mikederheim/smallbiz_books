Smallbooks visual refresh update

No MySQL changes are required.

Install:
1. Back up your existing Smallbooks files.
2. Copy the app/ and assets/ folders from this update over your existing installation.
3. In Safari, Chrome, or mobile browsers, hard-refresh the page after copying files.
   Safari may otherwise keep the old CSS cached.

Changed:
- Smaller top-left Smallbooks logo.
- Darker, cleaner sidebar closer to the mockup.
- White top header bar with company and logged-in user info.
- Cleaner Inter font stack loaded from Google Fonts with system fallback.
- More polished dashboard cards, tables, buttons, forms, and report headers.
- Dashboard welcome header closer to the mockup.
- Mobile menu behavior preserved.
