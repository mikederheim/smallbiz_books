SmallBiz Books Update: User Management

This is a changed-files-only update.

Install steps:
1. Back up your current smallbiz_books folder and database.
2. Copy the contents of this update folder over your existing smallbiz_books installation, preserving the same folder structure.
3. No MySQL changes are required.
4. Log in and use Admin > Users in the left sidebar.

What changed:
- The sidebar item is now "Users" instead of "Add User".
- Users screen lists existing users.
- You can add, edit, and remove users.
- Passwords remain hashed using PHP password_hash().
- Editing a user can leave the password blank to keep the current password.
- Safety rules prevent deleting the only user account or the account you are currently logged in as.
- Public registration remains available only when no users exist yet, for first-time setup.
