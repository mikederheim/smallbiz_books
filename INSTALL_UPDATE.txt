SmallBiz Books - Safari Desktop Menu Button Fix

This is a changed-files-only update.

Files changed:
- assets/css/style.css

Install:
1. Back up your current site files.
2. Copy the included assets/css/style.css over your existing file.
3. No MySQL changes are required.
4. In Safari, hard refresh the page or clear cached website data if the old CSS remains visible.

What changed:
- The mobile menu button is now explicitly hidden by default.
- It is forced hidden at desktop widths using a min-width media query.
- It is only shown at mobile widths of 800px or below.
